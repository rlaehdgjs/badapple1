블록 퍼즐 프로젝트 요구사항 (최종 확정안)
1. 목적
10x10 격자에서 콤보 배수 시스템을 활용해 최고 점수에 도전하는 전략 퍼즐 게임 구현.

2. 필수 기능
격자 및 블록: 10x10 보드, 매회 3개 블록 생성, 드래그 앤 드롭 배치.

[확정] 콤보 및 점수 로직:

콤보 조건: 블록을 배치했을 때 최소 1줄 이상 지워지면 Combo Count가 1 증가.

콤보 유지: 지워진 줄의 개수(1줄이든 3줄이든)와 상관없이 '연속으로' 줄을 지우기만 하면 콤보 유지.

콤보 초기화: 블록을 놓았는데 지워진 줄이 하나도 없다면 Combo Count는 즉시 0으로 리셋.

점수 계산 공식: 기본 점수 × (1.5 ^ Combo Count).

예: 1콤보 시 1.5배, 2콤보 시 2.25배, 3콤보 시 3.375배...

라인 클리어: 가로/세로 완성 줄 동시 제거 및 애니메이션.

게임 오버: 보드 내 블록 배치 불가 시 종료 및 LocalStorage에 최고 점수 갱신.

3. 디자인 요구
콤보 배수 표시: 현재 적용 중인 배수(예: "x1.5", "x2.25")를 점수판 옆에 강조하여 표시.

시각 효과: 콤보 수가 높아질수록 줄이 터지는 이펙트의 색상이나 크기를 화려하게 변경.

인터페이스: 10x10 보드 상단에 Score/Best Score, 하단에 블록 대기열 배치.

4. 기술 스택
Frontend: HTML5 (Canvas 또는 DOM 제어), CSS3, JavaScript (ES6+).

Server: Node.js (Express) 기반 정적 웹 서버.http://localhost:3000
